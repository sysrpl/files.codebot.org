This is the login page for Cloud File Manager, a free open source source project for cloud file storage. Anyone can freely reuse this project to create their own secure prersonal cloud file storage services. You can invite anyone to share access to files anywhere, using any device, for any purpose, be it to managing content for websites, backing up content to the cloud, or just for sharing files with friends over the Internet.

Cloud File Manager is different than other cloud file storage services you may have already used. When compared to Dropbox it doesn't require software to be installed each device when you want to backup or share files. Also you don't need to keep files on your device after they've been copied to the cloud. When compared to Google Drive it allows for custom storage locations, such as to your own personal file server or any third party storage service such as Amazon Simple Storage Service or Google Cloud Storage. In any case, you get to choose how and where your cloud files are stored.

When using this project to store website content you can directly link to it without restrictions, for example at any place you might use <audio>, <video>, and <img> markup. Whereas with the free cloud storage services, directly linking to you files is both blocked and prohibited.

Features include multiple simultaneous uploads with cancel, pause and resume for broken connections. Files can be renamed, deleted, and shared publicly. Public files can be distributed either as direct links or as self contained media player pages.
